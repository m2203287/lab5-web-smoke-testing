# Smoke Checklist

Сервис: `https://www.saucedemo.com/`

1. Стартовая страница открывается без ошибки.
2. Поле `user-name` доступно для ввода.
3. Поле `password` доступно для ввода.
4. Кнопка `Login` кликабельна.
5. После входа открывается страница `inventory.html`.
6. Карточки товаров отображаются корректно.
7. Кнопка `Add to cart` работает.
8. Badge корзины меняется на `1`.
9. Страница прокручивается к футеру по локатору.
10. Скриншоты сохраняются в `artifacts/`.
